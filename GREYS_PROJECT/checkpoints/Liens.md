# Grey's Project

## Les checkpoints

Les checkpoints sont des fichiers de sauvegardes .PTH qui enregistrent tous les paramètres du model pour une certaine base de donnée.

Vous aurez besoins des checkpoints pour effectuer des analyses. Vous pouvez créer vos propres checkpoints en lançant un entrainement depuis l'interface sur la/les bases de données voulu. 

Nous vous proposons nos checkpoints : 

- Cellules sanguines : [checkpoint_blood_cells.pth](https://www.kaggle.com/dorianea/checkpoint-blood-cells/download/Rju2HYiOoypdmOzoG4Q7%2Fversions%2FCgiZNJekUQGYVrGMrSoT%2Ffiles%2Fcheckpoint_blood_cells.pth?datasetVersionNumber=1)
- Tumeur cérébrale (IRM) : [checkpoint_brain_tumor.pth](https://www.kaggle.com/dorianea/checkpoint-braintumor/download/SH4YSHEmt25T8ThryppZ%2Fversions%2FmjJrng31sF9TexVt200D%2Ffiles%2Fcheckpoint_brain_tumor.pth?datasetVersionNumber=1)
- Melanome : [checkpoint_BD_melanome.pth](https://www.kaggle.com/theormmm/checkpoint-melanome/download/CFNW4YSb2ua27gKmDpzI%2Fversions%2FBdRkQ7lOBWiny1Yg95eS%2Ffiles%2Fcheckpoint_BD_melanome.pth?datasetVersionNumber=3)
